- 数据库：物理操作系统文件或其他形式文件类型的集合

- 实例：MySQL数据库由后台线程以及一个共享内存区组成

  数据库实例才是真正用于操作数据库文件  实例与数据库一一对应（MySQL是数据库实例）所以MySQL连接本质是进程间通信

MySQL数据库独有的**插件式存储引擎**（基于表），可移植

## 事务

### 4大特性：

**原子性、一致性、隔离性、持久性**

**实现保证：**

​		MySQL的存储引擎InnoDB使用重做日志保证一致性与持久性，回滚日志保证原子性，使用各种锁来保证隔离性。

### 隔离级别



# InnoDB存储引擎

支持`事务`，行锁设计、支持`外键`、默认读取操作不产生锁

多版本并发控制（	MVCC）获得高并发性

默认REPEATABLE级别  使用next-key避免幻读

提供插入缓冲、二次写、自适应哈希索引、预读

聚焦方式 （主键存放）

索引与数据一起

# MyISAM存储引擎

不支持事务，不支持行锁，只支持表锁不支持外键，；支持全文索引；缓冲池只缓存索引文件

索引与数据分开

# NDB

集群存储引擎（可用性高） 数据放在内存中

# Memory

数据放在内存中（适合临时）  默认使用哈希索引
